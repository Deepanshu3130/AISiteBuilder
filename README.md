# 🚀 AI Site Generator

AI Site Generator is a full-stack SaaS application that allows users to generate complete, production-ready websites using AI. Users can describe their idea, and the platform automatically generates a responsive website using modern web technologies.

---

## 🌟 Features

- ✨ AI-powered website generation (HTML + Tailwind CSS)
- 🔐 Authentication system (Better Auth)
- 📁 Project management (create, view, update projects)
- 💬 Conversation history for each project
- ⚡ Live preview of generated websites
- 🌐 Community section to explore public projects
- 📥 Download generated code
- 📱 Fully responsive UI

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router
- Axios

### Backend
- Node.js
- Express.js
- Prisma ORM
- PostgreSQL (Neon DB)

### Authentication
- Better Auth

### AI Integration
- OpenRouter (Qwen, StepFun, GPT-OSS, etc.)

### Deployment
- Frontend: Render 
- Backend: Render
